# dzen-post-comments
SPA: Comments 

### Using techologies: 
* Turbo monorepo with
    * Nest.js
    * React.js
* WebSocket
* JWT Authentication
* MongoDB
* Amazon SQS, Lambda (for saving posts in DB)
* GraphQL

### How to start project without Docker
1. `npm install` from root directory
2. `npm run dev` from root directory
